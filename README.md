# LeetCode (基本素养)
### 1、两数之和
   给一个整数数组和一个目标值，找出数组中和为目标值的两个数，并返回数组下标。
   <br>解体思路：<br>1,暴力法（循环遍历，两层for循环，如果是3数之和，那就三层for循环）,暴力法的时间复杂度是o(n2)。<br>
             2, 哈希表法，目标数组存到哈希表里，时间复杂度为o(n),

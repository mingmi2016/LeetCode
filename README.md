### 20、有效的括号(valid Parentheses)
给定一个只包括(),{},[]的字符串，判断字符串是否有效。（假如包含字母什么的也行，预处理的时候只留括号，其余去除就行了呗）
<br>
这种程序反应一下，指定用栈来解决呗
遇到左边的括号就入栈，遇到右边的括号就出栈比较，如果栈有值又遍历到字符串末尾，则返回false
如果栈为空，遇到一个右括号，返回false
